User Guide


Data Source

1kGP SNP Browser supports the latest human genome assembly: the 1000 Genomes 30x on GRCh38, focusing on chromosome 22. More information on this dataset can be found by clicking here.

The SNP Browser results exclude the 698 related samples in the 30x dataset, focusing only on unrelated samples. The reasoning for this is because related samples have the same genetics and would give unclear summary statistical results. 1kGP SNP Browser only focuses on SNPs, therefore, the dataset used contains non-indels and excludes the indels, to also reduce the variation which can have an impact on the SNPs. 1kGP SNP Browser also focuses on biallelic sites, since human SNPs are more common to be biallelic than multi-allelic, and would obtain concordant results for the summary statistics.

1kGP SNP Browser contains a genotypes dataset, more information on the dataset can be found through this link: Genotypes Data. 1kGP SNP Browser also has a phased genotypes dataset which is used to calculate haplotype diversity, more information on this dataset can be found through this link: Phased Genotypes Data.




Input Format



1kGP SNP Browser accepts different forms of query input data. The general query which accepts the gene name, start and stop of the SNP position, and the rsID of the SNP. The population query allows the user to select which populations they want to investigate. The summary statistics query allows the users to select which summary statistic they want to calculate.
The queries for the input format are explained in more details below.

Users will have to input and specify their email addresses and a query ID, so they are able to keep their data download and make it easier for them to identify their saved query.

Users will also be able to create an account in 1kGP SNP Browser by adding an email address and password so when users input queries they will receive a notification of when it is done, in the case of the query being too long.

Users should only input a query that is 1Mb or less.


General Query

Users can specify one, or all, of either:

Gene Name: User should specify the desirable gene they want to investigate

rsID name: User should specify the position of the SNP (eg. rs1290354662)

Start position: User should specify the start position of the SNP they want to investigate

Stop position: User should specify the stop position of the SNP they want to investigate



Population Query

Users will be able to specify which population they want to select and find information on. In addition, to the option of also selecting another population where both of those populations will be compared when producing the summary statistics.

Users choice of populations are:

  • African (AFR): Within the dataset there is a total of 661 samples
  • European (EUR): Within the dataset there is a total of 503 samples
  • East Asian (EAS): Within the dataset there is a total of 504 samples
  • American (AMR): Within the dataset there is a total of 347 samples
  • South Asian (SAS): Within the dataset there is a total of 489 samples


Summary Statistics Query

Users will be able to specify which summary statistics they want to view after specifying the general queries and selecting which population they want

Users choice of summary statistics are:

  • Tajima's D
  • Fst values
  • Genetic Diversity
  • Haplotype Diversity
  • Derived Allele Frequency

Summary Statistics

1kGP SNP Browser provides users with a set of summary statistics that they are able to select which ones they want. When users specify the summary statistics they want, 1kGP SNP Browser will return a set of values from each summary statistic and present a sliding window for them too.

The summary statistics that 1kGP SNP Browser will be returning for the user are:




Genotype Frequency

1kGP SNP Browser provides the users with the genotype frequency of each individual population. Genotype frequency is the prevalence or ratio of genotypes in a given population. It ranges from 0 < f < 1. When focusing on one population the genotype frequency is the number of individuals from a certain genotype divided by the total number of individuals in the specified population.


Allele Frequency

1kGP SNP Browser provides the users with the allele frequency of each individual population. Allele frequency is the prevalence of a gene variant within a given population. Allele frequency is measured by dividing the number of times the given allele is presented in a population by the total number of copies of all the alleles at that specific genetic location in the selected population.


Tajima's D

1kGP SNP Browser provides the users with the statistical test for neutrality (Tajima’s D) for comparing two of the populations selected by the users. Tajima’s D is a type of test that differentiates a randomly evolving DNA sequence between one that is non-randomly evolving. Tajima’s D is mainly used to identify sequences that do not apply to the neutral theory model at equilibrium due to mutation-drift.


Fst Value

1kGP SNP Browser provides the users with Fst values for each of the populations that the users have selected., also presenting them as a sliding window. Fst refers to the ratio of the total genetic variance embedded in a subpopulation. Fst values range from 0 to 1.


Genetic Diversity

1kGP SNP Browser provides the user with the genetic diversity of the individual population that has been specified. Genetic diversity makes reference to the different attributes a species has inherited, by focusing on the variations there are on the alleles. To calculate the genetic diversity in a population, 1kGP SNP Browser uses the Watterson estimator/Nucleotide diversity(?).


Haplotype Diversity

1kGP SNP Browser provides the users with the haplotype diversity for the selected population. Haplotype diversity makes reference to the probability that two randomly sampled alleles are different between each other, it ranges from 0 to 1.


Derived Allele Frequency

1kGP SNP Browser provides the user with another statistical test for neutrality (Derived Allele Frequency). Derived Allele Frequency (DAF) refers to the frequency that a mutated allele is present in a population.


Interpretation of Summary Statistic Results

1kGP SNP Browser has a set of summary statistics that the users can select their preferred ones.
The interpretation of these results are explained below.



Genotype Frequency

There are three different genotype frequency values for each of the 5 different populations: genotype frequency for homozygous reference allele, genotype frequency for homozygous alternate allele, genotype frequency for heterozygous alleles. All values range from 0 to 1.

High values of genotype frequency for heterozygous alleles within a population indicate that those alleles have increased heterozygosity in the population. High values of genotype frequency for homozygous reference alleles and homozygous alternate alleles within a population indicate that those alleles (reference and/or alternate) have increased homozygosity in the population.

Low values of genotype frequency for heterozygous alleles within a population indicate that there is less heterozygosity in the alleles, but there is more homozygosity in the alleles in the population. Low values of genotype frequency for homozygous reference alleles and homozygous alternate alleles within the population indicate that those alleles (reference and/or alternate) have less homozygosity in the alleles, but there is increased heterozygosity in the alleles in the population.


Allele Frequency

Allele frequency can provide information if there is any genetic diversity within a population. If there is a change in the allele frequency it can illustrate that there is a genetic drift taking place or that there is the presence of new mutations in the population.

A high value in allele frequency indicates that there is more variation within the population. Therefore, there is a high chance that a mutation has occurred somewhere in the human lineage and is found in the majority of the population.

A low value in allele frequency indicates that there is less variation within the population. Therefore, this means that a mutation has occurred in the human lineage but it is only shown in the minority of the population.


Tajima's D

The values for tajima’s D can be both negative and positive, depending on what the value is, it indicates different things.

If the Tajima’s D = 0 it indicates that the observed variation is complementary to the expected variation. Also, it means that there is no indication of selection, and that the population evolves in equilibrium due to mutation-drift.

If the Tajima’s D < 0 (negative) it indicates that there are less haplotypes, therefore decreasing heterozygosity. This implies that there is overabundance of rare alleles. Furthermore, it indicates an expansion in the population size and balancing selection.

If the Tajima’s D > 0 (positive) it indicates that there are additional haplotypes, therefore increasing heterozygosity. This implies that there is absence of rare alleles. Furthermore, it indicates a reduction in population size and balancing selection.


Fst Values

The Fst values as mentioned above range from 0 to 1. When the value is zero it is said to be a negative value, indicating that there is no genetic difference between the populations selected.

High Fst values indicate that there is a significant degree of genetic differentiation between the populations that have been selected by the users or that they have strong population subdivision.

Low Fst values indicate that there is absence of a significant degree of genetic differentiation between the selected populations by the users or lack of population subdivision.


Genetic Diversity

The higher the levels of genetic diversity there is within a population the higher the chance the individuals in that population will survive changes in the environment. As well as emerging diseases and/or epidemics. A high value of genetic diversity indicates that the individuals in the population present a broad array of distinct traits.

The lower the level of genetic diversity makes it harder for the individuals in the population to evolve, and decreases the likelihood of survival if there are any environmental changes. Low values of genetic diversity imply that the individuals in the population present a limited array of distinct traits.


Haplotype Diversity

Haplotype diversity represents a probability, and values range from 0 to 1.

High values of haplotype diversity indicate that there is a higher probability that the alleles within the population are different to each other. Indicating there is increased heterozygosity.

Low values of haplotype diversity indicate that there is reduced probability that the alleles within the population are similar to each other. Indicating there is increased homozygosity.


Derived Allele Frequency

Derived Allele Frequency (DAF) is a statistical distribution test for neutrality in population genetics, its values range from 0 to 1.

High value of DAF indicates that a mutation is present somewhere on the human lineage and it is identified in 95% of the humans from the population. These high values of DAF could be due to positive selection, weak balancing selection, likelihood allele will be fixed depending if the population size is low.

Low value of DAF indicates that a mutation is present in the human lineage but is only identified in 5% of the humans from the population.

Output Format

1kGP SNP Browser presents the summary statistics results from what the user has specified in the queries in a table format. Some of the summary statistics (Tajima’s D, Fst Values, Genetic Diversity) are also presented as a sliding window.

Each of the 3 sliding windows mentioned above will have the option to select 2 populations to compare, in the case that the users specify for more than 2 populations in the first population query.

1kGP SNP Browser also allows the users to specify how many base pairs (bp) the windows for the sliding window should be.

If the users have queried for more than one SNP, the table will provide each individual SNP.

A table will be presented with the Fst values produced from the comparison between each population the users have specified.

A table will be displayed with each of the 5 populations (AMR, EAS, SAS, EUR, and AFR) as rows, and will contain the values of each population of the following summary statistics as columns:

  • Nucleotide diversity as genetic diversity
  • Tajima's D
  • Haplotype diversity

Three tables will be presented showing the other summary statistics: Allele Frequency, Genotype Frequency, and Derived Allele Frequency. Each of these summary statistics are separated into tabs.


The table for Allele frequency will contain the following columns:

  • CHROM = The chromosome number
  • POS = The position of the SNP or of each SNP the users have queried
  • REF = The reference alleles of the SNP or of each SNP the users have queried
  • ALT = The alternate alleles of the SNP or of each SNP the users have queried
  • GENE = The gene name the users want to query
  • RS_VAL = The rsID for the SNP
  • AA = Ancestral allele
  • AF_AFR = Allele frequency from African population.
  • AF_AMR = Allele frequency from American population.
  • AF_EAS = Allele frequency from East Asian population.
  • AF_EUR = Allele frequency from European population.
  • AF_SAS = Allele frequency from South Asian population.

The table for Derived allele frequency will contain the following columns:

  • CHROM = The chromosome number
  • POS = The position of the SNP or of each SNP the users have queried
  • REF = The reference alleles of the SNP or of each SNP the users have queried
  • ALT = The alternate alleles of the SNP or of each SNP the users have queried
  • GENE = The gene name the users want to query
  • RS_VAL = The rsID for the SNP
  • AA = Ancestral allele
  • DAF_AFR = Derived allele frequency from African population.v
  • DAF_AMR = Derived allele frequency from American population.
  • DAF_EAS = Derived allele frequency from East Asian population.
  • DAF_EUR = Derived allele frequency from European population.
  • DAF_SAS = Derived allele frequency from South Asian population.

The table for Genotype frequency will contain the following columns:

  • CHROM = The chromosome number
  • POS = The position of the SNP or of each SNP the users have queried
  • REF = The reference alleles of the SNP or of each SNP the users have queried
  • ALT = The alternate alleles of the SNP or of each SNP the users have queried
  • GENE = The gene name the users want to query
  • RS_VAL = The rsID for the SNP
  • AA = Ancestral allele
  • GF_HET_AFR = Genotype frequency of heterozygous African population.
  • GF_HOM_REF_AFR = Genotype frequency of homozygous reference alleles in African population.
  • GF_HOM_ALT_AFR = Genotype frequency of homozygous alternate alleles in African population.
  • GF_HET_AMR = Genotype frequency of heterozygous American population.
  • GF_HOM_REF_AMR = Genotype frequency of homozygous reference alleles in American population.
  • GF_HOM_ALT_AMR = Genotype frequency of homozygous alternate alleles in American population.
  • GF_HET_EAS = Genotype frequency of heterozygous East Asian population.
  • GF_HOM_REF_EAS = Genotype frequency of homozygous reference alleles in East Asian population.
  • GF_HOM_ALT_EAS = Genotype frequency of homozygous alternate alleles in East Asian population.
  • GF_HET_EUR = Genotype frequency of heterozygous European population.
  • GF_HOM_REF_EUR = Genotype frequency of homozygous reference alleles in European population.
  • GF_HOM_ALT_EUR = Genotype frequency of homozygous alternate alleles in European population.
  • GF_HET_SAS = Genotype frequency of heterozygous South Asian population.
  • GF_HOM_REF_SAS = Genotype frequency of homozygous reference alleles in South Asian population.
